# Calculadora-em-Lua---N-vel-simples
Essa calculadora foi meu primeiro projeto, na linguagem que comecei a programar, Lua, espero que gostem.
Para funcionar, você precisa ter Lua 5.4.2 instalado no seu computador e colocar no Path do seu computador (windows).

**Linux/Mac:** Adicione o caminho no seu arquivo `.bashrc` ou `.zshrc`.

**Windows** Aperte windows + R, abra sysdm.cpl, aperte em avançados, procure variáveis de ambiente, procure PATH e quando achar clique uma, duas vezes ou clique em PATH e clique em editar, após isso cole o que estava escrito nso arquivos, exemplo: C:\computadordejunior\Junior\Desktop\Lua, após isso, clique em ok nas 3 janelas que abriram, depois abra o VS Code (se estiver aprendendo Lua baixe a extensão Lua de sumneko e quebre o código pra aprender a corrigir erros).

Após isso você tem duas opções, copiar o código e colar no Vs Code (jeito mais rápido).
ou
Baixar um dowload ou clone do Arquivo e colocar a pasta no VS Code (a pasta com o arquivo do código).
Bem, é isso, assim que se usa o meu código, se forem copiar e colar o código e usar em algum projeto por favor dar os créditos, se for baixar já vem com a licença então já vem com os créditos, até o próximo repositório e tchau.

OBS: as letras com acentos podem causar bugs no terminal e um "é" aparecer um símbolo tipo uma bola, se isso aconter retire os acentos das palavras que não irá mais bugar.

OBS 2: a calculadora suporta adição, subtração, multiplicação e divisão, ná próxima atualização do repositório irá ter suporte para porcentagem.
